# 个人待办事项页面

一个纯前端个人待办事项页面，使用 HTML、CSS、JavaScript 和 localStorage 实现。

## 已实现

- 快速添加任务，支持 Enter 提交
- 添加任务时选择高 / 中 / 低优先级，默认中优先级
- 添加任务时必须选择截止日期，并在任务列表展示
- 任务列表展示、完成状态切换、删除任务
- 简单的行内编辑和取消编辑
- 全部 / 未完成 / 已完成筛选
- 按未完成优先、优先级高低排序展示
- localStorage 本地保存，刷新后数据保留
- 空状态提示
- 桌面端和移动端基础响应式布局

## 未实现

- 任务搜索
- 分类管理
- 拖拽排序
- 任务备注
- 数据统计
- 深色模式
- 后端同步和登录
